Instalando dependencias:

express:
	- Framework para hacer API Rest sobre NodeJS, trabaja con protocolo HTTP.
	
bcript-nodejs:
	- Libreria para encriptar contrasenas, las pass se guardan encriptadas.
	
body-parser:
	- Parsea las peticiones recibidas por POST, parsealo a Jason y todos los datos recibidos se convierten en objetos.
	
connect-multiparty:
	- Permite subir ficheros por files, utilizando peticiones http (con NodeJS), y los formularios pueden tener un multi party y enviar datos a la API
	  guardandose en el servidor.

jwt-simple:
	- Libreria para la autentificacion con tokens.

moment:
	- Parse, validate, manipulate, and display dates and times in JavaScript.

mongoose:
	- Networking library with functions including TCP, HTTP client and server, WebSocket client and server, MQTT client and broker and much more.
	
mongoose-pagination:
	- Hacer paginaciones.

nodemon:
	- Is a utility that will monitor for any changes in your source and automatically restart your server. Perfect for development
